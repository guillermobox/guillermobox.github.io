The 6502 processor emulator
First steps

In order to emulate the 6502 processor family, we have to first know all that is possible about the technology itself. To do this, we can use the MOS manual for programmers that has been recently liberated. If we manage to complain with evey chapter and section of that manual, our system should behave exactly as it is supposed, from the point of view of the programmer. Hence, the applications and games developed for the processor should work as expected. This is a 2DIP encapsulation of the MOS6502:

Considering the following equation: $$f(a,b) = \int_0^\infty\mkern-3mm x^a \sin(bx) dx,$$ and then try to calculate the value of \(f\) for different values of \(a\) and \(b\), but without using the available computational resources in the cluster. Instead, try to relate the solution to a well known problem of integration, and give a symbolic representation of the integral.
